## Development Setup

### Prerequisites

- Install Node.js which includes Node Package Manager npm

### Setting Up a Project

Install the Angular CLI globally:

```
npm install -g @angular/cli
```

Install packages:

```
npm install
```


Run the application:

```
cd [PROJECT NAME]
ng serve
```
## RUN
